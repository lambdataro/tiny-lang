# tiny-lang

小さい自作言語の処理系

## 環境

- Windows 10
- CMake 3.14.2
- mingw-64 8.1.0

## コンパイル

```sh
$ cmake .
$ make
```

## テスト

```sh
$ ctest
```

## 文法

```
プログラム ::= 文; 文; ...
文 ::= 変数 = 式
    |  { 文; 文; ... }
    |  while (式) 文
    |  if (式) 文
    |  print(式)
式 ::= 変数
    |  整数
    |  文字列
    |  式 < 式
    |  式 + 式
    |  式 - 式
    |  式 * 式
    |  式 / 式
    |  ( 式 )
```
